# Overview
- A data analysis and predictive modeling project designed to assist a Portuguese banking institution in improving its direct marketing campaigns.
- The system analyzes customer demographics and economic -indicators to predict term deposit subscriptions and provides actionable strategies to increase conversion rates.

# Key Features

- Predictive Modeling: Built a Random Forest Classifier to predict customer subscription likelihood.
- Leakage Prevention: Developed a realistic model by excluding call duration (a future variable) to ensure real-world applicability.
- Strategic Insights: Identified key subscriber demographics (e.g., retirees, students) and optimal contact months to maximize ROI.
- Imbalance Handling: addressed significant class imbalance (11% success rate) to improve model recall.
